## 📌 Sobre o Projeto
Este é um sistema simples desenvolvido para a disciplina de Desenvolvimento de Sistemas (DS). O objetivo é ajudar usuários a entenderem quanto cada aparelho eletrônico gasta por mês na conta de luz.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python
* **Conceitos:** Entrada de dados, variáveis, operações matemáticas e f-strings para formatação.

## 🧮 Fórmula Matemática
Para chegar ao resultado, o programa utiliza o seguinte cálculo:

$$consumoMensal = \frac{potencia \times horasDia \times 30}{1000}$$

*Onde 1000 é usado para converter Watts em kWh.*

1. Certifique-se de ter o **Python** instalado no seu computador.
2. Baixe o arquivo `app.py`.
3. Abra o terminal ou prompt de comando na pasta do arquivo.
4. Digite o comando:
